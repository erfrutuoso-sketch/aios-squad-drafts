Política de Privacidade — AIOS Squad Drafts

Última atualização: 16/02/2026

1. Visão geral

O AIOS Squad Drafts é uma ferramenta local executada no computador do usuário para gerar vídeos e, mediante autorização, enviar vídeos ao Inbox do TikTok usando APIs oficiais. Esta política descreve como os dados são tratados.

2. Dados que coletamos

  • Tokens de acesso do TikTok (OAuth): armazenados localmente no computador do usuário.
  • Arquivos de mídia (fotos/vídeos) fornecidos pelo usuário: usados apenas para gerar o vídeo localmente.
  • Metadados locais (ex.: logs de execução): armazenados localmente para diagnóstico.

3. Dados que NÃO coletamos

  • Não vendemos dados.
  • Não coletamos dados sensíveis intencionalmente.
  • Não compartilhamos tokens com terceiros; eles ficam no dispositivo do usuário.

4. Como usamos os dados

  • Tokens: autenticar solicitações ao TikTok para enviar o vídeo ao Inbox.
  • Mídia: compor e renderizar o vídeo localmente.
  • Logs: ajudar na solução de problemas técnicos.

5. Compartilhamento

Não compartilhamos dados com terceiros. O envio ao TikTok ocorre somente quando o usuário autoriza e executa o fluxo de upload.

6. Retenção e segurança

Os dados ficam armazenados localmente no computador do usuário. O usuário pode apagar a pasta do projeto e os arquivos de token para remover os dados. Recomendamos manter o computador protegido por senha.

7. Seus direitos

O usuário pode revogar permissões do TikTok a qualquer momento pelas configurações da conta TikTok e pode excluir arquivos locais (tokens, logs e mídias) no próprio dispositivo.

8. Contato

Para dúvidas, utilize o canal de contato informado no repositório/website do projeto.
